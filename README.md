# nullsid-sniper
basic nullsid sniper, letting you get id as soon as it is dropped.

## Requirements:
* zig 0.16.0+
* Any text editor

## Instructions on using
1. Fill in **handle** and **bearerstr** variables in src/main.zig _(to find bearer use [this project](https://github.com/RandomName132333/nulls-connect-client))_
2. Go to the build.zig location and enter in your console:
```
zig build --release=fast
```
3. Launch exetuble using
```
zig-out/bin/g
```
